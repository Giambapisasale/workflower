Interroga il sistema

Linguaggio naturale → il Query Agent genera SQL sulle viste DuckDB (v_fatture, v_ddt, v_ore, v_voci_preventivo…) che leggono i file JSON

SQL generato tier T2 · viste: v_fatture_righe ⋈ v_materiali ⋈ v_voci_preventivo
SELECT m.descrizione, SUM(r.quantita) AS qta, SUM(r.importo) AS costo,
       p.importo AS preventivo, SUM(r.importo) - p.importo AS scostamento
FROM v_fatture_righe r
JOIN v_materiali m        ON m.id = r.materiale_id
JOIN v_voci_preventivo p  ON p.voce_id = r.voce_computo_id
WHERE r.cantiere_id = 'CNT-001' AND m.categoria = 'calcestruzzo'
  AND r.data BETWEEN '2026-01-01' AND '2026-12-31'
GROUP BY 1, 4;
